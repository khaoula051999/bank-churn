Bienvenue à l'application de prédiction de Churn

Cette application utilise l'intelligence artificielle pour prédire le churn des clients bancaires. Vous pouvez utiliser le simulateur pour entrer les détails d'un client et obtenir une prédiction, ou consulter les notifications pour les mises à jour importantes.
